# SMH Trading Website

## Selwa Muhammed Hussain Al Omari Est

Welcome to the official source code for the SMH (Selwa Muhammed Hussain Al Omari Est) retail website. SMH is a trusted retail shop specializing in high-quality electrical, plumbing, and building materials.

This is a modern, responsive, and bilingual (English and Arabic) static website designed to showcase their catalog of trade supplies, detailed category overviews, company details, and contacts.

## ✨ Features

- **Bilingual Interface**: Seamlessly toggles between English (LTR) and Arabic (RTL) using vanilla Javascript. 
- **Responsive Layout**: Designed with a mobile-first approach, ensuring a clean and legible interface across mobile devices, tablets, and massive desktop displays.
- **Modern UI/UX**: Built with a clear navigation bar, sticky headers, floating WhatsApp integration, feature highlights, and highly segmented product categories.
- **Fast & Pure**: Built strictly on fundamental web technologies. Without bulky frontend frameworks, it optimizes directly for speed, minimalism, and broad support.

## 🛠️ Tech Stack

- **HTML5**: Built with semantic markup and embedded bilingual text structures.
- **CSS3**: Crafted utilizing CSS Variables natively, incorporating Flexbox, CSS Grid matrices, hover logic, and robust RTL styling. 
- **JavaScript (Vanilla)**: Orchestrates robust UI functionalities such as smooth scrolling, the off-canvas mobile menu navigation, and the DOM-altering bi-directional language switch.
- **Font Awesome 6**: Imported for reliable, stylistic, and scalable vector iconography.
- **Google Fonts**: Uses **Inter** for English typography and **Tajawal** for seamless Arabic display. 

## 🚀 Getting Started

To view, modify or contribute to this project locally, start a local HTTP server ensuring asset paths map correctly. It has no external build processes, so there are zero compiling steps!

### Prerequisites

You can use any generic HTTP server such as `http-server` via npm, the Visual Studio Code "Live Server" extension, or simply use Python 3's built-in web server.

### Quick Start

1. Clone this repository directly onto your local environment:
   ```bash
   git clone <your-repository-url>
   cd smh
   ```

2. Start a local HTTP server. For instance, using Python 3:
   ```bash
   python3 -m http.server 8080
   ```

3. Launch your browser and navigate to:
   ```text
   http://localhost:8080
   ```
